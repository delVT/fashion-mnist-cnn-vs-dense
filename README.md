# Fashion-MNIST: CNN vs Dense Networks

Comparison of a fully connected neural network and a convolutional neural network 
for image classification using the Fashion-MNIST dataset. The project demonstrates why
CNN architectures are better suited for image data.

## Tools
Python, TensorFlow/Keras, NumPy, Matplotlib

## Concepts Demonstrated
- Image classification
- Neural network architectures
- Convolutional layers and pooling
- Feature extraction and spatial learning
- Model performance and error analysis
